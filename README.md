## TOpoli ##

Visualizing Toronto's political and demographic data.

##### Getting started (OSX): #####

Install mongodb:

```
brew install mongo
```

Start mongo

```
mongod --dbpath ~/mongodata
```

Fire up the app
```
npm install
npm start
```
